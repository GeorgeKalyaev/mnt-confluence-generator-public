# Как получить trial-лицензию для Confluence Server

## ⚠️ Важно:
**Поддержка Confluence Server завершилась 15 февраля 2024 года.** Но для тестирования trial-лицензия все еще может работать.

## Пошаговая инструкция:

### Шаг 1: Получите Server ID из локального Confluence
1. Откройте http://localhost:8090
2. На странице ввода лицензии вы увидите **Server ID** (например: `BOSZ-DKX4-P9ON-HM5Y`)
3. **Скопируйте этот Server ID** - он понадобится для получения лицензии

### Шаг 2: Откройте страницу лицензий
Перейдите на: **https://my.atlassian.com/**

### Шаг 3: Войдите в аккаунт
- Войдите используя тот же аккаунт, которым вы авторизовались в Confluence Cloud

### Шаг 4: Создайте новую trial-лицензию
1. Нажмите кнопку **"New Trial License"** (Новая пробная лицензия)
2. Или перейдите: **https://my.atlassian.com/licenses/new**

### Шаг 5: Выберите продукт
1. В списке продуктов найдите **"Confluence Server"** (НЕ Cloud!)
2. Убедитесь, что выбрали именно **Server**, а не **Cloud**

### Шаг 6: Заполните форму
1. **Server ID:** Вставьте скопированный Server ID из шага 1
2. **Organization name:** Название вашей организации (можно любое для теста)
3. **Organization size:** Выберите подходящий размер (например, "1-10 users")
4. Нажмите **"Generate license"** или **"Get license"**

### Шаг 7: Скопируйте лицензионный ключ
- После создания вы получите **License Key** (длинная строка)
- Скопируйте его полностью

### Шаг 8: Введите ключ в локальный Confluence
1. Вернитесь на http://localhost:8090
2. Вставьте скопированный лицензионный ключ
3. Нажмите **"Next"** или **"Continue"**

## Альтернативный способ (если не нашли опцию):

### Вариант A: Через форму запроса
1. Перейдите: **https://www.atlassian.com/software/confluence/try**
2. Заполните форму
3. Выберите **"Server"** (не Cloud)
4. Получите лицензию

### Вариант B: Прямая ссылка на Server trial
Попробуйте: **https://www.atlassian.com/software/confluence/download**

## Важно:
- Лицензия для **Confluence Server** отличается от **Confluence Cloud**
- Убедитесь, что выбрали именно **Server** версию
- Trial лицензия действует 30 дней

## Если возникли проблемы:
- Убедитесь, что вы на странице для Server, а не Cloud
- Попробуйте использовать другой браузер
- Очистите кэш браузера
