# Changelog

Все значимые изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-01-18

### Добавлено
- Упрощенная конфигурация: все настройки в `app/core/config.py`
- Автоматические бэкапы базы данных (ежедневно в 02:00)
- Централизованная обработка ошибок
- Валидация и санитизация входных данных
- Логирование действий безопасности
- Экспорт всех данных в архив
- Восстановление из бэкапа
- Улучшенная обработка ошибок при публикации в Confluence
- Поддержка предупреждений в UI (вместо ошибок)

### Изменено
- Убрана поддержка `.env` файлов - все настройки в `app/core/config.py`
- Рефакторинг структуры проекта: разделение на модули (`core/`, `routes/`, `services/`, `utils/`, `middleware/`)
- Разделение роутов на отдельные модули (`main.py`, `mnt.py`, `api.py`, `admin.py`) с использованием `APIRouter`
- Вынос middleware в отдельный модуль (`middleware/logging.py`)
- Улучшена структура проекта для лучшей организации кода
- Обновлена документация с актуальной структурой проекта

### Исправлено
- Исправлен редирект при создании МНТ без настроенного Confluence
- Исправлена проверка конфигурации Confluence
