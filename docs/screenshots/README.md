# Скриншоты интерфейса МНТ Confluence Generator

Эта папка содержит скриншоты интерфейса приложения МНТ Confluence Generator.

## Доступные скриншоты

1. **Список МНТ.png** - Главная страница со списком всех документов МНТ
2. **Создать новый МНТ.png** - Форма создания нового документа МНТ
3. **Публикация в Confluence.png** - Интерфейс публикации документа в Confluence
4. **История версий МНТ.png** - Страница просмотра истории версий документа
5. **Сравнение версий МНТ.png** - Интерфейс сравнения двух версий документа
6. **Корзина.png** - Интерфейс корзины для восстановления удаленных документов

## Использование

Эти скриншоты используются в основном файле [README.md](../../README.md) для демонстрации возможностей приложения.
