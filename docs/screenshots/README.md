# Скриншоты проекта

Эта папка предназначена для размещения скриншотов интерфейса проекта МНТ Confluence Generator.

## Рекомендуемые скриншоты для добавления:

1. **list-page.png** - Главная страница со списком МНТ
2. **create-page.png** - Форма создания нового МНТ
3. **edit-page.png** - Форма редактирования МНТ
4. **table-editors.png** - Визуальные редакторы таблиц
5. **confluence-publish.png** - Интерфейс публикации в Confluence

## Требования к изображениям:

- **Формат:** PNG или JPG
- **Размер:** Рекомендуется ширина 1200-1920px
- **Имя файла:** Используйте понятные имена (list-page.png, create-page.png, и т.д.)

После добавления изображений обновите ссылки в главном `README.md` если нужно.
